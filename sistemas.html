Sistemas Operativos

Un sistema operativo (SO) es software que gestiona hardware y recursos para ejecutar programas. Facilita interacción usuario-computadora y administración tareas. Aquí hay 15 términos clave:

Kernel:

Núcleo central del SO, controla hardware y servicios fundamentales.

Interfaz Gráfica de Usuario (GUI):

Entorno visual que simplifica la interacción con el sistema.

Procesos:

Programas en ejecución, gestionados por el SO.

Memoria RAM:

Espacio temporal para datos y programas activos.

Sistema de Archivos:

Estructura organizada para almacenar y recuperar datos.

Multitarea:

Ejecución simultánea de varios procesos.

Multihilos:

División de un proceso en subprocesos para mejorar el rendimiento.

Seguridad:

Protección contra accesos no autorizados y amenazas.

Red:

Facilita la comunicación entre dispositivos en una red.

Controladores (Drivers):

Software que permite la comunicación entre hardware y SO.

Sistema de Archivos Virtual:

Abstracción para facilitar el acceso a archivos.

Shell:

Interfaz de línea de comandos para interactuar con el SO.

Página:

Unidad de transferencia de datos entre memoria principal y secundaria.

Planificador (Scheduler):

Asigna recursos y tiempo de CPU a procesos.

Sistema de Archivos Distribuido:

Almacena datos en varias ubicaciones para redundancia y eficiencia.
