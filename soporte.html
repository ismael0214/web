Diccionario Digital

Soporte Técnico

El soporte técnico es el servicio que provee una empresa a sus clientes, con el fin de que reciban ayuda acerca del uso de sus productos, ya sean físicos o digitales. Puede estar proporcionado directamente por la compañía o estar a manos de agentes especializados externos.

Auth-Code:

También conocido como código EPP o Auth-Info, el Auth-Code es un código de autorización necesario para la transferencia de un dominio genérico.

Backup:

También llamada copia de seguridad o respaldo, es una copia de toda la información de un sitio web.

DNS:

El sistema de nombres de dominio (Domain Name System), se encarga de traducir a palabras los números que componen una dirección IP.

.htaccess:

El archivo .htaccess es un fichero de configuración de directorios que podrás encontrar en servidores web Apache.

Webmail:

O correo web, es una interfaz que permite el acceso a un buzón de correo de forma remota sin necesidad de utilizar un cliente de correo.

Whois:

Protocolo de transmisión (TCP) que actúa como base de datos en la que se registra información relacionada con el titular de un dominio.

WPO:

El Web Performance Optimización, u optimización del rendimiento web, está compuesto por una serie de técnicas que permiten mejorar la velocidad de carga de un sitio web.

Backbone:

Columna vertebral de la red.

Joystick:

Periférico en forma de palanca y con botones incorporado, diseñado especialmente para disfrutar de los videojuegos.

Proxi:

Servidor que realiza la conexión a Internet y que sirve de puerta de entrada a los ordenadores cliente.

Token Ring:

Protocolo para redes de IBM que consiste en un anillo donde una especie de relevo se encarga de “pasar turno” entre las máquinas que desean transmitir la información.

Upload:

Subir un fichero a un servidor de Internet.

Tracert:

Comando que analiza y registra el camino que ha seguido cualquier información enviada por la red.

Yahoo:

Portal más popular de Internet que permite la búsqueda de páginas web por criterios o contenidos.
